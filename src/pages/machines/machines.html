MACHINES

How easy do you think it will be if you are to uncork a mineral or nail a wood with your bare hands? Machines make it easier when carrying out some of these tasks. By definition, machine is a device that helps us to do work more easily.


The energy put into a machine for it to do work is called work input and the work done by the machine is called work output.


Work output is always less than the work input because some amount of the input will be used to overcome friction which will generate heat, at times sound and/or light (fire). It therefore follows that, the efficiency of a machine is always less than hundred percent (100%). Efficiency is a measure of how effective a machine, system or a process is.

Efficiency (ƞ) is defined as:

ƞ = work output

Classification of Machines

Machines can be grouped into 6 categories. They are

  1. Lever

  2. Pulleys

  3. Screw

  4. Wheel and axle

  5. Inclined plane and wedge

  6. Hydraulic press


The lever

The lever is a rigid bar which turns freely around a fixed point known as pivot or fulcrum.

There are three classes of lever


  1. First class levers; this has the pivot between the load and the effort (See picture below).


  2. Second class levers; this has the load between the pivot and the effort. (See picture below).
